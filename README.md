

### **Setup**

1. **Clone the Repository**:
   - git clone https://github.com/ShwetaEchake/family.git
   - cd family

2. **Install Dependencies: Install PHP and Laravel dependencies**:
    - composer install

3. **Set Up Environment Variables**:
    - you can copy .env.example file and create a .env file , also i will share the file via email as well.
    - update the database credentials in the .env file.
    - DB_DATABASE=family_registry
    

4. **Run Migrations: Migrate the database to create necessary tables**:
    - php artisan migrate

5. **Storage Link:**:
    - php artisan storage:link


6. **Start the Development Server: Run the Laravel development server**:
    - php artisan serve
